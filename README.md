This repo is the free companion to the **Blue Team Field Manual (BTFM)** — practical scripts, cheatsheets, and field content released openly as a contribution back to the community that made the book possible.

The book gives you the doctrine. This repo gives you the tools to run it.

> Written by [Alan White](https://www.linkedin.com/in/alan-j-white) | [BTFM on Amazon](https://www.amazon.com/dp/B077WF4WYV)

## Quick Start

```bash
git clone https://github.com/aw31337/btfm.git
cd btfm
```

Scripts are organized by phase. Navigate to the relevant folder, read the header comments, and run. Most bash scripts require standard Linux tools; PowerShell scripts require PS 5.1+ on Windows.
